python环境搭建

发表于 2018-05-24

Python基础（一）环境搭建

环境说明：

操作系统：windows7
python版本：3.6

正文：

安装

  首选是安装包，访问python官网，选择好自己想要的版本以及符合自己电脑的版本下载即可。

1.我选择的是：Windows x86-64 executable installer（windows64位系统使用安装程序），选择定制安装

2.第二步，全部勾选，目前只了解一个pip是用于管理python工具类库的工具，其他暂时还未用到

3.第三步，勾选的是一个关联所有.py的文件，一个是创建快捷方式，一个是创建环境变量，未勾选的一个是为所有用户安装，一个是安装预编译标准库（保存代码时会提前编译，执行的时候稍微快些），最后两个是debug相关的，不怎么了解。

环境变量

  环境变量主要是一个path路径，将python的安装路径和安装路径下的Scripts文件夹路径都加进去，路径之间用半角字符分号隔开，作用是系统调用python库时，系统会按path路径一个个寻找，找到执行库时才能执行。如果使用exe安装程序勾选了Add python to PATH 就不需要配置环境变量，自动添加在当前用户的全局变量中。
例如我的环境变量是：

C:\Users\Administrator\AppData\Local\Programs\Python\Python36\Scripts\;
C:\Users\Administrator\AppData\Local\Programs\Python\Python36\;
